# Usage
<!-- usage -->
```sh-session
$ npm install -g kickoff
$ kickoff COMMAND
running command...
$ kickoff (-v|--version|version)
kickoff/0.0.0 linux-x64 node-v12.16.1
$ kickoff --help [COMMAND]
USAGE
  $ kickoff COMMAND
...
```
